## Brief Overview

This project presents a comparative study of Field-Oriented Control and Direct Torque Control for electric motor drive applications. Both methods are commonly used for high-performance AC motor control.

Field-Oriented Control uses dq-axis current control to independently control flux and torque, resulting in smooth response and low torque ripple. Direct Torque Control directly controls torque and flux using hysteresis controllers and voltage vector selection, providing faster dynamic response but higher torque ripple.

The comparison is carried out based on torque response, flux control, current behavior, dynamic performance, switching behavior, implementation complexity, and suitability for electric drive applications.
